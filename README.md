# Projet Semestre
## Setup
### Linux
`sudo make install-dependencies`

`make setup`

### Windows
Installer npm et node.js (https://github.com/coreybutler/nvm-windows), et python (pour SQLite3)

## Build
### Linux
`make build`

### Windows
`npx electron-forge import`

`npm run make`
